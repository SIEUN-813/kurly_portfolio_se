### Branch 사용하기
    1. git branch > 초기 브랜치
    => $ git branch
        * master

    2. git branch 생성되는브랜치이름 > 브랜치 생성
        git branch love

    3. git branch > 브랜치 생성 확인
    => $ git branch
        love
        * master

    4. git checkout 사용할브랜치이름 > 브랜치 변경
        git checkout love

    5. git branch > 브랜치 변경 확인
    => $ git branch
        * love
        master
